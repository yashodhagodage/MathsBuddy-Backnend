MathsBuddy API Documentation

END-POINT:

https://maths-buddy.herokuapp.com/


User Routes

# Path Method Description
1 /user GET Get all users
2 /user/register POST Register a user
3 /user/:id PUT Update user by _Id
4 /user/login POST User Login
5 /user/:id GET Get user by _id
6 /user/delete/:id DELETE Delete user by _id

Question Routes

# Path Method Description
1 /question GET Get all Questions
2 /question/create POST Create a Question
3 /question/:id PUT Update By ID
4 /question/:id GET Get By ID
5 /user/login POST User Login
6 /question/getcategory/:category GET Get Questions by Category
7 /question/:id DELETE Delete question by _id

Favorite Routes

# Path Method Description
1 /favorite GET Get all Favorites
2 /favorite/add POST Add/Create a Favorite
3 /favorite/byid/:id GET Get favorite by _id
4 /favorite/byuser/:user_id GET Get favorites of an user by user_id
5 /favorite/:id DELETE Delete by a favorite by _id
